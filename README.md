# 🛒 Online Shopping Cart System

---

## 📖 Description

An interactive command-line based **Online Shopping Cart System** developed in Python using Object-Oriented Programming (OOP) concepts.  
This project simulates a simple e-commerce platform where users can browse products, select brands, choose product categories, and add items to their shopping cart.

---

## 🚀 Features

- User registration and login system
- Interactive menu-driven interface
- Multiple shopping categories:
  - Fashion
  - Mobile
  - Laptop
  - Vehicle
  - Food
  - Book
- Brand-based product selection
- Product pricing display
- Add-to-cart simulation
- Built using Python classes and inheritance

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes & Objects
- Inheritance
- Functions & Methods
- Conditional Statements
- User Input Handling
